# План розробки Telegram-бота

## Крок 1 — Підготовка середовища
- Встановити Python (3.10+)
- Створити віртуальне середовище (`venv`)
- Встановити залежності (`pip install python-telegram-bot` або `aiogram`) + зберегти в `requirements.txt`
- Створити бота через @BotFather, отримати токен
- Створити `.env` з токеном, `.gitignore`
- Визначити структуру папок проекту (`handlers/`, `keyboards/`, `db/` тощо)
- Ініціалізувати git-репозиторій, зробити перший коміт

## Крок 2 — Базова структура бота
- Команда `/start` + головне меню з кнопками
- Навігація між розділами

## Крок 3 — Екстрена кнопка *(пріоритет 1)*
- Кнопка "Потрібна допомога прямо зараз"
- Надсилає пріоритетний запит менеджеру

## Крок 4 — Каталог послуг та товарів *(пріоритет 1)*
- Список послуг з описом та ціною
- Товари по категоріях: фото, опис, ціна, позначка "на замовлення"
- Якщо товару немає — кнопки "Написати менеджеру" і "Зателефонувати"

## Крок 5 — Пакетні пропозиції *(пріоритет 1)*
- Пакети "Економ", "Стандарт", "Преміум"

## Крок 6 — Оформлення замовлення *(пріоритет 1)*
- Збір даних: ім'я, телефон, населений пункт
- Згода на обробку персональних даних
- Підтвердження з номером заявки

## Крок 7 — Сповіщення менеджеру *(пріоритет 1)*
- Надсилання заявки в окремий чат/канал
- Можливість скасування клієнтом

## Крок 8 — FAQ *(пріоритет 2)*
- Відповіді на типові запитання без участі менеджера

## Крок 9 — Статус замовлення *(пріоритет 2)*
- Перевірка статусу клієнтом
- Сповіщення при зміні етапу

## Крок 10 — Робочі години
- Логіка відповіді поза робочим часом (8:00–23:00)

## Крок 11 — Адмін-панель
- Керування каталогом без програміста (додати / змінити / прибрати товар)
- Перегляд історії всіх заявок
- Налаштування робочих годин через бота

## Крок 12 — Тестування та деплой
- Тести основних сценаріїв
- Деплой на сервер (Railway, VPS тощо)
